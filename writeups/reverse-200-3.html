The Glass Bead Game

Challenge Description

Multi-layer encoding with base64 and ROT13.

Solution

The binary uses base64 encoding followed by ROT13. Reversing both reveals the flag.

$ echo "encoded" | base64 -d | tr 'A-Za-z' 'N-ZA-Mn-za-m'
