Challenge Description

XOR encoding challenge requiring basic cryptanalysis.

Solution

The binary uses XOR encoding. Analyzing with Ghidra reveals the XOR key.

$ python3 -c "print(''.join(chr(ord(c) ^ 0x42) for c in encrypted))"
